Une landing page moderne construite avec Next.js et React, stylée en CSS classique (sans Tailwind) avec prise en charge du thème clair/sombre/système.

## Aperçu
- Thème clair/sombre/système (JS + variables CSS)
- Menu burger mobile animé + panneau déroulant
- Navigation avec survol soigné
- Footer simple

## Démarrage
- Développement: `npm run dev`
- Build: `npm run build`
- Production: `npm start`

## Structure
- Code applicatif: `app/`, `components/`
- Styles globaux: `app/globals.css`
- Script de thème: `public/theme.js`
